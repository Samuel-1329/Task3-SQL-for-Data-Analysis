## 💡 Interview Questions Preparation

### Difference between WHERE and HAVING?

* WHERE filters rows before aggregation.
* HAVING filters groups after aggregation.

### Types of Joins

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN

### What are Subqueries?

A query written inside another query.

### What is a View?

A virtual table based on the result of a SQL query.

### How do you handle NULL values?

Using functions like IFNULL(), COALESCE(), or conditions with IS NULL.

### How do you optimize SQL queries?

* Create indexes
* Avoid unnecessary columns
* Use efficient joins
* Optimize filtering conditions
